# Healthcare Data Analytics Dashboard

## Project Overview
This project analyzes hospital patient records using Power BI and SQL to generate insights related to patient admissions, treatment costs, and discharge trends.

## Technologies Used
- Power BI
- SQL
- Excel / CSV
- Power Query

## Features
- Interactive Healthcare Dashboard
- Patient Admission Analysis
- Treatment Cost Insights
- Discharge Trend Monitoring
- Department-wise performance analysis

## KPIs Created
- Total Patients
- Admission Rate
- Average Treatment Cost
- Discharge Trends

## Business Impact
- Reduced manual reporting effort by 40%
- Improved reporting efficiency by 30%
- Enabled data-driven decision making
